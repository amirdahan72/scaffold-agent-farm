# Diagram Prompt: GitHub Copilot Agent Mode Architecture

> **Use this prompt in M365 Creator (or any AI image/diagram tool) to generate a professional visual for Slide 3.**

## Prompt

Create a professional, clean architecture diagram titled **"GitHub Copilot Agent Mode — How It Works"** using a modern tech-corporate style with a white background and blue/teal accent colors.

### Layout

**Center**: A large rounded rectangle labeled **"VS Code + GitHub Copilot Chat (Agent Mode)"** — this is the main container. Use a soft blue (#0078D4) border and light blue fill (#F0F7FF).

**Inside the container, show 3 horizontal layers:**

1. **Top layer — "PM Interaction"**: A person icon labeled "PM" on the left, with a speech bubble showing "Natural language prompt". An arrow flows right into a box labeled "Agent (defined by .agent.md)".

2. **Middle layer — "Agent Capabilities"**: Show 5 small rounded boxes in a row:
   - "Read/Write Files" (folder icon)
   - "Run Terminal" (terminal icon)
   - "Web Search" (globe icon)
   - "MCP Servers" (plug icon)
   - "Sub-Agents" (people icon)

3. **Bottom layer — "Tool Connections"**: Show 4 external service boxes connected by dashed arrows from the MCP Servers box:
   - "Azure Resources" (cloud icon, Azure blue #0078D4)
   - "Azure DevOps" (DevOps icon, blue)
   - "Work IQ / M365" (M365 icon, orange #E97132)
   - "Web" (globe icon, teal #467886)

**Right side outside container**: Show output boxes:
- ".md files" 
- ".docx files"
- ".pptx files"
- ".xlsx files"
- "Charts (PNG/SVG)"

### Style
- Clean, flat design — no 3D, no gradients
- Segoe UI or similar sans-serif font
- Color palette: Navy (#0E2841), Azure Blue (#0078D4), Teal (#467886), Orange (#E97132), White, Light Gray (#E8E8E8)
- 16:9 aspect ratio
- Arrows should be clean with subtle arrow heads
- Add subtle drop shadows on the main container only
