# Diagram Prompt: Data Sources & Output Ecosystem

> **Use this prompt in M365 Creator (or any AI image/diagram tool) to generate a professional visual for Slide 9.**

## Prompt

Create a professional **hub-and-spoke diagram** titled **"Agent Farm Data Ecosystem — Sources & Outputs"** showing how Agent Farms integrate multiple data sources and produce multiple output formats. White background, modern tech-corporate style.

### Layout — Hub and Spoke

**Center Hub**: A large hexagon or rounded rectangle labeled **"Agent Farm"** with sub-text "Orchestrator + Sub-Agents". Use Navy (#0E2841) fill with white text. Inside, show small icons for the 5 agent types in a row.

**Left Side — "Data Sources" (Inputs)**: 5 spoke connections flowing INTO the hub from the left:

1. **Web Search** (Globe icon, Teal #467886)
   - Sub-text: "Competitor sites, analyst reports, news, pricing"
   - Arrow flowing right into hub

2. **Work IQ / M365** (M365 icon, Orange #E97132)
   - Sub-text: "Emails, Teams chats, meetings, SharePoint docs"
   - Arrow flowing right into hub

3. **Azure MCP** (Cloud icon, Azure Blue #0078D4)
   - Sub-text: "Resource health, deployments, App Insights, Monitor"
   - Arrow flowing right into hub

4. **Azure DevOps** (DevOps icon, Blue #156082)
   - Sub-text: "Work items, sprint backlogs, iteration status"
   - Arrow flowing right into hub

5. **PM Resources** (Document icon, Medium Gray #747474)
   - Sub-text: "Existing specs, roadmaps, compete docs"
   - Arrow flowing right into hub

**Right Side — "Output Formats"**: 6 spoke connections flowing OUT of the hub to the right:

1. **Markdown (.md)** (Text icon)
   - Sub-text: "PRDs, briefs, scorecards, narratives"

2. **Word (.docx)** (Word icon, Blue)
   - Sub-text: "Stakeholder sharing, formal documents"

3. **PowerPoint (.pptx)** (PPT icon, Orange)
   - Sub-text: "Leadership reviews, analyst briefings"

4. **Excel (.xlsx)** (Excel icon, Green)
   - Sub-text: "Feature matrices, trackers, scorecards"

5. **Charts (PNG/SVG)** (Chart icon, Teal)
   - Sub-text: "Bar, line, pie, heatmap visualizations"

6. **Run Archives** (Folder icon, Gray)
   - Sub-text: "Versioned runs with all intermediate artifacts"

### Key Annotations
- Below the diagram: "All outputs are local files — version-controlled, auditable, and shareable"
- Input arrows should be solid lines; output arrows should also be solid but slightly different style (perhaps thicker)
- Each source/output should have a small colored icon matching its theme

### Style
- Clean, flat design — no 3D, no gradients
- Segoe UI or similar sans-serif font
- The hub should be the visual focal point (largest element)
- Inputs on left, outputs on right — clear directional flow
- Color palette: Navy (#0E2841), Azure Blue (#0078D4), Teal (#467886), Orange (#E97132), Green (#107C10), Light Gray (#E8E8E8)
- 16:9 aspect ratio
- Use subtle color-coding to match each source/output to its brand color
