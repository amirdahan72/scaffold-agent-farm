# Diagram Prompt: The Agent Farm Pipeline (6 Phases)

> **Use this prompt in M365 Creator (or any AI image/diagram tool) to generate a professional visual for Slide 6.**

## Prompt

Create a professional **horizontal pipeline/flow diagram** titled **"The Agent Farm Pipeline — 6 Phases"** showing how a PM deliverable moves through 6 automated phases. White background, modern tech-corporate style.

### Layout — Horizontal Flow (Left to Right)

Show 6 connected phases as rounded rectangles in a flowing pipeline. Each phase connects to the next with a prominent arrow. Between certain phases, show a small diamond labeled "PM Checkpoint" in Azure Blue.

**Phase 0 — "Resource Gate"** (Gray, #747474)
- Icon: 📋 clipboard
- Label: "PM drops docs & links"
- Sub-text: "Agent asks for approval"
- Tag: "PM action"
- → PM Checkpoint diamond →

**Phase 1 — "Collection"** (Teal, #467886)
- Icon: 🔍 magnifying glass
- Label: "Parallel data gathering"
- Show 3 small boxes inside: "Web Researcher", "WorkIQ Collector", "Resource Reader"
- Sub-text: "Each writes to sources/*.md"
- Tag: "Automated"
- → PM Checkpoint diamond →

**Phase 2 — "Synthesis"** (Teal, #467886)
- Icon: 🔀 merge
- Label: "Merge all sources"
- Sub-text: "Produces combined-draft.md"
- Tag: "Automated"
- → PM Checkpoint diamond →

**Phase 3 — "Skeptic Review"** (Coral/Red, #F65567)
- Icon: 🔎 critical eye
- Label: "Adversarial review"
- Sub-text: "Finds unsupported claims, bias, gaps"
- Tag: "Automated"
- → PM Checkpoint diamond →

**Phase 4 — "Revision"** (Teal, #467886)
- Icon: ✏️ pencil
- Label: "Fix every issue"
- Sub-text: "Produces revised-draft.md with change log"
- Tag: "Automated"

**Phase 5 — "Final Output"** (Teal, #467886)
- Icon: 📦 package
- Label: "Polished deliverable"
- Show output format icons: .md, .docx, .pptx, .xlsx
- Tag: "Automated"

### Key Visual Elements
- Below the pipeline, add a callout box: "Each sub-agent gets a fresh context window — no overflow from accumulated research"
- The PM Checkpoint diamonds should stand out in Azure Blue (#0078D4) with a small person icon
- Phase 3 (Skeptic) should visually stand out as different — use the coral/red color to signal "adversarial"

### Style
- Clean, flat design — no 3D
- Segoe UI or similar sans-serif font
- Color palette: Navy (#0E2841), Teal (#467886), Coral (#F65567), Azure Blue (#0078D4), Light Gray (#E8E8E8)
- 16:9 aspect ratio
- Horizontal flow, phases should read left to right
- Pipeline arrows should be thick and clear
